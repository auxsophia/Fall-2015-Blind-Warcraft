Thank you for downloading Blind Warcraft

Installation Instructions

1. Unzip blind-warcraft-releaseV01.zip
2. Place the BlindWarcraft folder in your addons directory as you would any other wow addon.
3. Place the blindwow.py file in your NVDA global plugins directory.  This should be stored in Users/AppData/Roaming/nvda
4. Go to www.jocys.com/Software and download the Text to Speech Monitor.
5. Install that like any other wow addon.
6. Replace the original JocysCom executable with the modified one included in the BlindWarcraft release

Using Blind Warcraft

Restart NVDA to make sure the plugin is loaded
Run the executable from the JocysCom directory, and start playing!
